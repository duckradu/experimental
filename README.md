# Kameleon Monorepo

## Local setup

### Global dependencies

- node (>= 20)
- [pnpm](https://pnpm.io/installation) ([alias setup (optional)](https://pnpm.io/installation#using-a-shorter-alias))
- Docker

### Install dependencies

Run `pnpm install` at the root of the monorepo (where this README.md is located).
